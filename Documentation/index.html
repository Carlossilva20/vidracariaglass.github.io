Getting Started

What's Included

1. HTML - Once you have download the package you will see the following folder structure in "HTML" folder. This folder contains static HTML CSS files

  • css - Main css files
  • img - Image files
  • js - Custom js files
  • plugins - All used plugins

 

2. SCSS - You will see the following folder structure in "SCSS" folder. This folder contains SCSS files if you want to use Sass as a css preprocessor.

  • scss - All Scss files
  • css - Output css from Scss files
  • img - Image files
  • js - Custom js files
  • plugins - All used plugins

HTML Structure

Angel template is based on Bootstrap Framework (http://getbootstrap.com/) Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.

Below is a sample coding structure:

                <!DOCTYPE html>

                <html lang="en-US">

                <head>

                

                </head>

                <div class="main-wrapper">

                  </header>

                    <div class="container">

                      ...

                    </div>
                  </div>

                  
                  <footer>

                      ...

                  </footer>

                </div>

                </body>

                </html>

Layout Style

The template supports two different layout modes, Boxed & Wide Layout. The wide one is the default option, if you want to use the Boxed version, add the following class in the <body> and layout style will be changed to boxed version.

<body class="bodyColor wrapper">

Background Pattern:For Boxed version

if you want to use background pattern in the Boxed version, add one of the following class"pattern-default, pattern1, pattern2, pattern3" in the <body> like this:

<body class="bodyColor wrapper pattern1">

Color Schemes

This part of the documentation will explain how to change the current Color Scheme of your site, You simply need to include an existing color scheme or change the HEX Color Code in the css/colors/default.css and include inside <head>

Default color example


                <head>

                
                <link rel="stylesheet" href="css/colors/default.css" id="option_color">

                </head>


How to change default color:

To change default color, go to this foldercss/colors/default.css and open default.css.Change the HEX Color Code whatever you want like: color: #9b59b6;

Changing Fonts

If you wish to add/change the site fonts, please take a look in the head part of the website and replace this tag:


                
                <link href='https://fonts.googleapis.com/css?family=Oxygen:400,300,700' rel='stylesheet' type='text/css'>
